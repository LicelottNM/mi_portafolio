Tablero de Control para Monitoreo de Carga de Trabajo

Diseñé un tablero interactivo en Excel para visualizar métricas operativas en tiempo real, identificar cuellos de botella y redistribuir tareas, optimizando la eficiencia de los equipos.

Dashboard de Ventas Mensuales con Datos Limitados

Diseñé un dashboard interactivo a partir de un conjunto reducido de datos de ventas mensuales, enfocándome en representar de forma clara los ingresos, tendencias y variaciones por categoría. El proyecto desafió mi creatividad y capacidad de análisis para generar valor con recursos limitados.

Modelado de Relaciones con Neo4j

Diseñé y manipulé una base de datos orientada a grafos con Neo4j para representar relaciones complejas entre entidades. Utilicé consultas Cypher para analizar patrones de conexión y optimizar la estructura de los datos.

Análisis de Mercado para el sector entretenimiento

Proyecto en desarrollo pensado para ser un analisis de mercado sobre consumidores del cine dominicano, usando power bi y tecnicas de web Scrapping en python.
